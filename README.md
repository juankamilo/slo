#Proyecto para validar superficies limitadoras de obstaculos

Tenemos un archivo xsd en resources/AIXM_Features.xsd con las definiciones de un archivo xml que debemos generar

Requerimientos
Generar Interfaces del xsd para validar con typescript los datos
Generar una base de datos en sqlite, los campos deben ser en ingles, por mas que te defina los nombres en espanol
Debe contar con una tabla aeropuertos (airport) con id y name. creaer 3 (arauca, saravena, ocana)
Crear una tabla obstaculos esta tabla debe tener los campos (id, obstaculo, tipo obstaculo, area, altitud, norte, este, id_airport)
